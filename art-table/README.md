# Artwork Gallery

A React application with a data table that displays artwork data from the Art Institute of Chicago API with server-side pagination and persistent row selection.

## Features

- ✅ **Server-side pagination** - Fetches data per page from API
- ✅ **Persistent row selection** - Maintains selections across page navigation
- ✅ **Custom selection panel** - Select specific number of rows
- ✅ **Multiple selection modes**:
  - Individual row selection
  - Select all/deselect all on current page
  - Global "select all" mode
  - Custom N-row selection
- ✅ **TypeScript** - Fully typed components
- ✅ **Responsive design** - Works on mobile and desktop
- ✅ **No mass data storage** - Only stores current page data
- ✅ **No prefetching** - Compliant with assignment requirements

## Technology Stack

- **React 18** with TypeScript
- **Vite** for build tooling
- **PrimeReact** for UI components
- **LocalStorage** for selection persistence
